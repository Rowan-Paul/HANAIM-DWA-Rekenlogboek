## Logbook

**`POST`** `/logbook`

Create a new logbook.

| Parameters | Send in body                  | Return as JSON |
| ---------- | ----------------------------- | -------------- |
| None       | JSON object with logbook info |                |

---

**`PUT`** `/logbook/:id/currentPhase`

Updates the currentPhase for a logbook.

| Parameters     | Send in body                  | Return as JSON |
| -------------- | ----------------------------- | -------------- |
| id - logbookid | JSON object with currentPhase |                |

---
