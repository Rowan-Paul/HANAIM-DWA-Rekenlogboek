# Teamafspraken

Om het project soepel te laten verlopen zijn de volgende afspraken gemaakt door het team:

- Ons doel is een 6,5
- Code is in het Engels, documentatie en interface in het Nederlands (waarschijnlijk)
- Er wordt gewerkt vanuit de ontwikkeltak, waarbij elke uitgave een aparte tak krijgt (met recensies voor het samenvoegen)
- Het gebruik van de camera is geen vereiste, behalve tijdens een ontmoeting met PO of coach op Teams
- Als er iets misgaat en je kunt het niet redden, stuur dan een bericht naar het team op Discord
- Als er conflicten zijn, heeft de scrummaster het laatste woord

## Dagelijkse planning

Wij werken elke werkdag van 9.00 tot 17.00 uur. Dagelijkse stand-up wordt gehouden aan het begin van de dag, om 9.00 uur. De teamleden kunnen een pauze nemen wanneer ze willen, zolang ze de rest van het team maar een bericht sturen over onenigheid. De lunchpauze is van 12.00 tot 12.30 uur.

## Software

De volgende software wordt door het hele team gebruikt:

- ESlint / Prettier voor code-opmaak
- Visual Studio Code voor codering
- Discord voor teamvergaderingen
